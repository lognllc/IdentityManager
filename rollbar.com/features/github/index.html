Github Integration

If you host your code at Github and connect your Github account with Rollbar, you'll be able to:

  • log into Rollbar with Github oauth
  • have filenames in stack traces linked to the corresponding file and line in Github
  • create Github Issues from Rollbar items
  • see the list of commits in each deploy
  • in some cases, Rollbar can guess which commits caused a particular error (requires deploy tracking)

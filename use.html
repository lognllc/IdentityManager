Frameworks & Languages


Spring is the most popular application development framework for enterprise Java™.

Java is a programming language and computing platform that runs on billions of devices worldwide.

Ruby is a dynamic, open source programming language with a focus on simplicity & productivity.

Ruby on Rails is an open-source web framework that's optimized for programmer happiness and sustainable productivity.

Sinatra is a DSL for quickly creating web applications in Ruby with minimal effort.

Node is a platform built on Chrome's JavaScript runtime for easily building fast, scalable network applications.

Grails is an Open Source, full stack, web application framework for the JVM.

The Play framework makes it easier to build web applications with Java & Scala.

Services


Redis is an open source, advanced key-value store.

MySQL is the world's most popular open source database software, with over 100 million copies of its software downloaded or distributed throughout its history.

PostgreSQL is a powerful, open source object-relational database system with more than 15 years of active development.

RabbitMQ is a complete and highly reliable enterprise messaging system based on the emerging AMQP standard.

MongoDB is a scalable, high-performance, open source NoSQL database.
